# 《李敖对话录》思想索引校对说明

- 提取轮条目：133
- 校对轮条目：117
- 删除条目：16

## 校对原则

- 只处理条目取舍、标题、分类和关键词，不改写 description 原文。
- 删除同题重复、展开不足、偏自我机锋、偏专门例证和个案色彩过重的条目。
- 保留能够独立检索的写作方法、史料方法、知识分子伦理、文化判断、政治原则、法权意识、人格标准与情爱观段落。
- 标题继续压缩为检索用语，分类仍使用 8 个原子分类。

## 删除条目

- LAT056-006｜大向优先：偏苏洪月娇等党外个案攻防，原则性判断已由“大目标”“战斗标准”等条目承载，校对轮删去。
- LAT056-012｜知识起义：台湾知识分子逃避现实主题已由后文《花花公子》访问中的“知识人逃避”更完整承载，校对轮删去。
- LAT056-029｜力量尺度：以许荣淑、康宁祥为例说明力量尺度，个案色彩较重，独立索引价值不如领袖条件和压力政治条目，校对轮删去。
- LAT056-033｜西化事实：“西化是不可避免的事实”段落较短，同题由“西化即世界化”“读透后失望”等条目展开，校对轮删去。
- LAT056-035｜社会实验：“土城看守所是实验室”机锋较强但展开不足，社会研究方法已由长考、真相规则等条目承载，校对轮删去。
- LAT056-052｜小依大向：“小功夫”主题已由前一条集中呈现，本条主要补充大方向条件，校对轮删去。
- LAT056-055｜象征留史：立法院象征意义与保留历史记录的判断已由下一条“选举大目标”完整展开，校对轮删去。
- LAT056-065｜欺负回来：“把它欺负回来”偏个人快意和玩笑式反击，思想展开不如同书其他人格、政治条目稳定，校对轮删去。
- LAT056-067｜语音寻根：台湾话语音寻根偏专门训诂例证，和本书主干思想索引关系较旁，校对轮删去。
- LAT056-074｜会读古书：读中国书要会读的主题已由“古书定位”更清楚地说明，校对轮删去。
- LAT056-077｜制衡支持：支持反对势力以维持制衡的主题，与后文“制衡进步”“反对大向”重合，校对轮删去。
- LAT056-080｜内容能打：“内容能打”夹杂封面玩笑和市场趣味，写作出版方法由“写作工厂”“言论自由生态”承载，校对轮删去。
- LAT056-102｜信己所思：“信己所思”偏自我信念表达，方法论展开较薄，校对轮删去。
- LAT056-113｜无须领袖：党外无须领袖与前一条“山头策略”同题，且展开较少，校对轮删去。
- LAT056-118｜思想定位：“思想定位”是短句式自我定位，后文“思想探底”具体说明方法和限制，校对轮删去。
- LAT056-120｜现代文明：现代文明与全盘西化主题已由“西化即世界化”集中说明，校对轮删去。
