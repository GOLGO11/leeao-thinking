# 《李敖书序集续集》思想索引校对说明

- 提取轮条目：85

## 校对原则

- 只处理条目取舍、标题、分类和关键词，不改写 description 原文。
- 删除广告式简介、过渡短句、出版过程说明、外部引文占主体和同题重复的条目。
- 保留能独立检索的政治判断、法权意识、写作方法、史料方法、文化判断和人格伦理段落。
- 标题继续压缩为检索用语，分类仍使用 8 个原子分类。

## 删除条目

- LAT055-017｜人像历史：《历史与人像》简介偏广告式概括，段落短，独立思想展开不足，校对轮删去。
- LAT055-020｜文学成家：“文学成家”偏大全集宣传文案和自我定位，和本书中更具体的写作、历史、评论条目相比检索价值较弱，校对轮删去。
- LAT055-031｜革命狼狈：“革命狼狈”是《文茜半生缘》序中的过渡判断，篇幅短且依赖上下文，校对轮删去。
- LAT055-032｜个人失败：“个人失败”与后文“革命牺牲”“勇怯统独”主题重合，且段落过短，校对轮删去。
- LAT055-037｜消极不字：“消极不字”主要引出后文阉夫案，思想展开由“权力释放”承载，校对轮删去。
- LAT055-038｜阉夫杀夫：“阉夫杀夫”段落过短且停在引出外部结论，情爱判断不够完整，校对轮删去。
- LAT055-044｜文化疗伤：“文化疗伤”与下一条“文化本行”同题，后者展开更完整，校对轮删去。
- LAT055-046｜小岛国宝：“小岛国宝”外部引文占比高，李敖判断集中度不如“自由胜独”“文化本行”，校对轮删去。
- LAT055-050｜早死免见：“早死免见”机锋较强但偏文人美人闲论，和后续“自悔少作”“少作史料”相比索引价值较弱，校对轮删去。
- LAT055-055｜黑牢反扑：“黑牢反扑”主要交代《调查局研究》出版后的影响，法权思想由后文黑牢与刑求条目承载，校对轮删去。
- LAT055-056｜煽风出书：“煽风出书”偏出版缘起和自我调侃，思想展开不足，校对轮删去。
- LAT055-062｜破命系列：“破命系列”偏节目与合作缘起，校对轮保留对迷信体系和破除方法更集中的条目。
- LAT055-063｜挡术士财：“挡术士财”偏节目反馈和传播效果，思想判断不如“迷信欺民”“智慧破雾”集中，校对轮删去。
- LAT055-070｜拒迎权势：“拒迎权势”和下一条“良知挨打”同题，后者更能呈现坚持批判的代价，校对轮删去。
- LAT055-074｜对话留正：“对话留正”偏书籍推荐和内容介绍，政治判断不如“共同反独”“祯祥扫妖”明确，校对轮删去。
